# DevelopmentContributions are welcome! Please follow these guidelines:## Coding Standards- Use Python 3.8+- Follow PEP 8- Use type hints where helpful- Write docstrings for all public functions## Testing```bash# Run testspytest tests/# Run specific testpytest tests/test_model.py```##Submitting Changes1. Fork the repository2. Create feature branch3. Make changes with tests4. Submit pull request## Development Setup```bash# Clone and install in dev modegit clone https://github.com/yourusername/ffcnn.gitcd ffcnnpip install -e .# Install dev dependenciespip install -r requirements-dev.txt```
